helpCosa significano i termini del Content Lab
📊 PI (Performance Index)
= reach del post / reach mediana del profilo. Sopra 1× = sopra la media, sotto 1× = sotto media. ★ Top performer = post con PI ≥ 1,5×.
🧮 Reach mediana profilo
= valore "medio reale" che divide a metà tutti i post. Più affidabile della media aritmetica perché non influenzato dai picchi virali.
🎯 Top performer
= slot giorno+fascia oraria in cui hai pubblicato post con PI ≥ 1× (sopra mediana). Da replicare.
🧪 Slot da testare
= giorni+fasce orarie mai sperimentate. Prioritizzati su base benchmark esterno (mattine feriali, martedì/giovedì pomeriggio).
🔥/❄️ Indicatori formato
= best (verde) e weak (grigio) per la metrica nella tabella per formato.

Condividere la dashboard con altre persone

La dashboard è un file HTML standalone. Per condividerla hai 4 strade, dalla più semplice alla più strutturata:

1 · Mandare il file via email/WhatsApp
Salvi il file HTML e lo invii. La persona deve aprirlo nel browser e inserire la sua API key Windsor.ai. La key tua resta nel tuo browser e non viene condivisa.
2 · Caricarla su un drive condiviso (Google Drive, Dropbox, OneDrive)
Carichi il file e condividi il link di download. Stessa cosa: ogni persona inserisce la propria API key al primo accesso.
3 · Hostarla online (gratis) con GitHub Pages, Netlify Drop o Vercel
Trascini il file su netlify.com/drop e ottieni un URL pubblico tipo dashboard-epf.netlify.app. Chiunque apre l'URL vede la dashboard. Attenzione: chi non ha l'API key vedrà solo l'interfaccia vuota — devi condividere la key separatamente, o creare una versione "view-only" con dati hardcoded per stakeholder esterni.
4 · Dashboard Windsor.ai nativa (per stakeholder esterni)
Windsor.ai stessa offre dashboard condivisibili via link pubblico dalla loro web app. Soluzione più semplice se vuoi condividere con team o clienti senza dare l'API key a nessuno.
Sicurezza API key: non condividere mai la tua API key Windsor.ai in chat pubbliche, screenshot o file. Chiunque la possieda può vedere tutti i dati del tuo account Instagram. Se è stata esposta, rigenerala dalle impostazioni Windsor.ai.

Configura Windsor.ai

Inserisci la API key Windsor.ai del progetto. Chiedila in privato a Lapo — non viene mai condivisa pubblicamente. Resterà salvata solo nel localStorage del tuo browser, da inserire una volta per dispositivo.

Come ottenerla: Windsor.ai → Account → API access → copia il token.
